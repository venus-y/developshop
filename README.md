# 패션 플릭스 (Fashion Flicks)
개발기간 : 2023/11/7 ~ 2024/01/05

<br>

## 👗 온라인 의류 편집샵
다양한 브랜드가 입점한 온라인 편집샵 컨셉의 프로젝트입니다.
<br>

사용자들이 쉽게 원하는 의류나 브랜드를 찾을 수 있게 하는 것에 중점을 두었습니다.
<br>

## ⚙ 사용기술
- Backend: `Java 11` `Spring Framework` `MyBatis` `JSP`
- Frontend: `jQuery`
- DB: `MySql`
